# ToolSky - Premium Online Tools Collection

A professional website offering 13 free online tools including word counter, image resizer, background remover, QR generator, PDF tools, and more.

## Features

- 13 fully functional tool pages with front-end demos
- Responsive design (mobile, tablet, desktop)
- Smooth animations and hover effects (AOS library)
- SEO optimized with meta tags and Open Graph
- AdSense ready with placeholder sections
- Modern, professional gradient design
